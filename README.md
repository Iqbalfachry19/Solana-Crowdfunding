# Anchor Crowdfunding Program

Anchor-based implementation of the crowdfunding spec.

## Instructions

- `create_campaign(goal, deadline)`
- `contribute(amount)`
- `withdraw()`
- `refund()`

## PDAs

- Vault: `[b"vault", campaign]`
- Donation: `[b"donation", campaign, donor]`

## Build

```bash
anchor build
```
